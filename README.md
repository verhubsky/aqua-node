# AquaNode

Консольное приложение, моделирующее фрагмент работы цифрового хозяйства в аквакультуре. Главная задача - управление установками водоснабжения и кормушек для выращивания рыбы. Задача управляющего фермой поддерживать жизнедеятельность рыб, следить за состоянием бассейнов - контролировать качество воды, количество корма и уровень кислорода в воде.

---

## Быстрый Старт

### Установка

```bash
git clone https://github.com/verhubsky/aqua-node.git
cd aqua-node
```

### Запуск

```bash
python -m main
```
